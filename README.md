# deeplearning-courses
My repo for practicing/labs while learning from deeplearning.ia and fast.ai


## The Jekyll web site

Run it:

```sh
cd jekyll-gh-pages

# Only needed the very first time
bundle update

# Run the server and host your jekyll website locally
bundle exec jekyll serve
```
